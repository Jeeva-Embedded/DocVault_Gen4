1 What is the Gen4 Document Portal?

The Gen4 Document Management Portal is a private, internal web application built for the Gen4 Manufacturing team. It provides a single, organized place to store, search, and manage all technical and operational documents — engineering drawings, quality reports, software files, spreadsheets, images, and more.

Every file you upload is saved directly to the team's Google Drive folder for permanent storage, while the portal keeps a searchable index so the whole team can find documents instantly. Changes sync to everyone in real time — no refresh needed.

Key Benefits

  • Centralized document storage — one place for all departments
  • Instant search across file names, notes, and authors
  • Organized by category (Mechanical, Electronics, Software) and sub-categories
  • Full audit trail — who uploaded, who last modified, and when
  • Direct download from Google Drive without leaving the portal
  • Accessible from any device with a browser — no installation required
Technology The portal is built with React and backed by Google Firebase (real-time database) and Google Drive for file storage. It is hosted on GitHub Pages and requires no login — anyone with the URL can access it.

4 Uploading Files

Click Upload File in the sidebar to go to the Upload page. Files are uploaded directly to the Gen4 Google Drive folder.

1
Select files Drag and drop files onto the drop zone, or click the zone to open a file picker. Multiple files can be added at once.
2
Choose Category and Sub-Category Select the correct department category (e.g. Mechanical) and the relevant sub-category (e.g. QC).
3
Enter your name in the Author field (required) This records who uploaded the document. The field is mandatory — the upload button will not work without it.
4
Add optional notes Use the Description/Notes box to add any context — revision number, project reference, part number, etc.
5
Click Upload Each file shows a progress bar. A green tick appears when a file is saved to Drive. A success toast confirms all uploads completed.
Upload may take 10–30 seconds per file Files are sent to Google Drive via an Apps Script. Large files or a slow connection may take longer. Do not close the browser tab while uploading.
Tip — Remove a file before uploading Added a file by mistake? Click the X button next to it to remove it from the queue before pressing Upload.

5 All Documents Page

Click All Documents in the sidebar to see every document in the portal. Use the category tabs at the top to filter by department.

Category Tabs

The tab bar shows All, the three built-in categories (Mechanical Electronics Software), and any custom categories the admin has added. Click a tab to filter the document list to that category.

Sub-Category Tabs

Once you select a category, a second row of tabs appears for sub-categories (e.g. General, QC, IQC, and any custom ones). Click a sub-category tab to filter further.

Search

The search bar in the top-right filters documents in real time by file name, notes, or author name. Click the × inside the search bar to clear it.

Date Filter

Click the Filter button (next to the search bar) to reveal the date range filter. Enter a From Date, a To Date, or both to narrow results to a specific period. The Filter button turns green with a dot (●) when a date filter is active. Click Clear to remove all filters.

6 Viewing & Downloading a Document

View (Eye icon)

Clicking the eye icon opens a side panel on the right with the document's full details — name, category, sub-category, author, notes, file size, and dates. The panel also has View in Drive and Download buttons.

  • View in Drive — opens the file inside Google Drive in a new tab (great for previewing PDFs and images online)
  • Download — triggers a direct file download to your computer without going through Drive

Download (Download icon)

Clicking the download icon in the table row directly downloads the file to your device without opening the viewer panel. The portal records a download count each time this is used.

Tip Use View in Drive if you want to preview the file online. Use Download if you need to save it locally for offline use or editing.

7 Editing a Document Record

Click the pencil icon in the Actions column to open the Edit Document dialog. You can update the following fields:

  • Document Name — rename the file entry in the portal (does not rename the file in Drive)
  • Category — move the document to a different department
  • Sub-Category — change the sub-group
  • Author — the original uploader's name (required)
  • Modified By — enter your own name to record who made this edit (required)
  • Notes — update or add additional notes

Click Save to apply changes. The Modified Date column will automatically update to the current date and time.

Note Editing a document record updates the portal's index only. It does not alter the actual file stored in Google Drive.

8 Deleting a Document

Click the trash icon (red) in the Actions column. A confirmation dialog will appear. If you confirm:

  • The document record is removed from the portal index immediately
  • The portal attempts to delete the file from Google Drive as well
Permanent Action — Cannot Be Undone Once deleted, the document is permanently removed from both the portal and Google Drive. Be certain before confirming deletion.

Adding a Custom Category

Scroll to the bottom of the sidebar and click + Add Category.

1
Enter a category name E.g. "Quality Assurance", "Production", "Finance". An icon is generated automatically from the name.
2
Optionally add sub-categories Type a sub-category name and press Enter or click Add to include it.
3
Click Create Category The new category appears instantly in the sidebar, tab bar, and upload form.

Editing or Deleting a Custom Category

In the sidebar, hover over any custom category to reveal the pencil (edit) and trash (delete) icons. Editing allows you to rename the category and manage its sub-categories. Deleting a category does not delete the documents inside it — they remain in the portal under the old category name.

Adding Sub-Categories to Existing Categories

Go to All Documents and select a category tab. Click + Add Sub in the sub-category bar to create a new sub-category for that department. To remove a custom sub-category, click the small × next to its name in the sub-category bar.

10 Google Drive Files

Click Google Drive Files in the sidebar to see all documents that have a linked Google Drive file. This page shows only portal documents that are stored in Drive — it does not show your entire Drive.

From here you can view the Drive link, download directly, or open the document viewer for full details.

12 Best Practices

Always fill in the Author field Enter your real name (not a nickname or initials) so the team can trace documents back to the right person.
Use descriptive file names before uploading The portal uses the original file name. Rename files like Rev3_Chassis_Drawing_2026.pdf before uploading — it is much easier to search for later.
Add notes for context Use the Notes field for revision numbers, related project codes, part numbers, or any information that helps colleagues understand what the document is without opening it.
Update "Modified By" when editing Whenever you edit a document record, enter your name in the Modified By field so there is a clear audit trail of who changed what.
Do not use the Delete button unless necessary Deletion is permanent and removes the file from Google Drive. If you uploaded the wrong file, upload the correct one and then delete the wrong one.
